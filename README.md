# 💪 SMARTFIT GYM TRACKER

Una app web simple y fácil para registrar tus entrenamientos en el gimnasio SMARTFIT Colombia.

## 🎯 ¿Qué es?

**SMARTFIT GYM TRACKER** es una aplicación web minimalista diseñada para que registres rápidamente cada ejercicio que realizas en el gimnasio. Perfecto para llevar control de tu progreso sin complicaciones.

## 🚀 Características

✅ **Selector de máquinas pre-cargadas** - Todas las máquinas estándar de SMARTFIT  
✅ **Registro rápido** - Solo 4 datos: reps, sets, tiempo y calorías  
✅ **Estadísticas en tiempo real** - Ve tus totales mientras entrenas  
✅ **Historial del día** - Lista completa de ejercicios realizados  
✅ **Almacenamiento local** - Se guarda automáticamente en tu dispositivo  
✅ **Funciona offline** - Sin conexión a internet  
✅ **Responsive** - Funciona en celular, tablet y computadora  

## 🏋️ Máquinas Incluidas

- **Piernas:** Leg Press, Leg Extension, Leg Curl, Leg Press Hack
- **Pecho:** Chest Press, Chest Fly
- **Espalda:** Lat Pulldown, Seated Row, Back Extension
- **Hombros:** Shoulder Press
- **Brazos:** Tricep Dip, Bicep Curl
- **Abdominales:** Abdominal Crunch
- **Múltiple:** Smith Machine
- **Cardio:** Treadmill, Bicicleta Estática, Elíptica

## 📱 Cómo Usar

### En 3 pasos:

1. **Abre la app:** Haz clic en [MiEntrenamiento.html](https://kparedesc.github.io/smartfit-gym-tracker/MiEntrenamiento.html)

2. **Registra tu ejercicio:**
   - Selecciona la máquina
   - Ingresa repeticiones (reps)
   - Ingresa series (sets)
   - Ingresa tiempo en minutos
   - Ingresa calorías estimadas

3. **¡Listo!** 
   - El ejercicio aparece en la lista
   - Los totales se actualizan automáticamente
   - Se guarda en tu dispositivo

## 💾 ¿Dónde se guarda?

Los datos se guardan en el **almacenamiento local de tu dispositivo** (localStorage). Significa:

✅ **Pros:**
- Se guarda automáticamente
- No necesitas internet
- Es privado (solo tú ves tus datos)

⚠️ **Importante:**
- Si limpias el caché del navegador, se pierden los datos
- Los datos están solo en ese dispositivo
- Recomendamos tomar capturas de pantalla de tus logros

## 🎮 Controles

| Botón | Función |
|-------|---------|
| ✅ Agregar Ejercicio | Agrega el ejercicio a la lista |
| 🗑️ Limpiar todo | Borra todos los ejercicios del día (con confirmación) |
| X (en cada ejercicio) | Elimina ese ejercicio específico |

## 🎓 Tips para Mejor Uso

### 1. **Sé Consistente**
```
Registra cada ejercicio inmediatamente después de completarlo
```

### 2. **Estima Calorías**
```
Máquinas de resistencia: 80-150 cal por 20 minutos
Máquinas cardio: 150-300 cal según intensidad
```

### 3. **Progreso Semanal**
```
Aumenta 1-2 reps por semana en cada máquina
Registra el histórico para ver mejora
```

### 4. **Metas Realistas**
```
✓ Quemar 300+ calorías por sesión
✓ Entrenar 4-6 días por semana
✓ 45-60 minutos de sesión
✓ Aumentar fuerza consistentemente
```
